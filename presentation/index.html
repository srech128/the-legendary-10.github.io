Was ist jQuery?

  • freie JavaScript-Bibliothek
  • veröffentlicht 2006 von John Resig
  • meistverwendete Javascript-Bibliothek:
    • 2/3 der 10.000 meistbesuchten Websites nutzen jQuery

Warum jQuery?

  • DOM-Manipulation: Änderung einzelner Elemente (z.B. Farbe, Position etc.)
  • vorher: getElementById jetzt: $("#mydiv")
  • dynamische Website durch Ajax-Schnittstelle
  • durch zahlreiche Plug-Ins lassen sich erweiterte Funktionen hinzufügen, etwa ein erweitertes UI

Was ist jQuery Mobile?

  • Touch-optimierte JavaScript-Bibliothek
  • basiert auf jQuery
  • wurde 2010 veröffentlicht

Vorteile von jQuery Mobile

  • einheitliche Oberfläche auf allen unterstützten mobilen Plattformen
  • lightweight und performant
  • Dateigröße nur 351kB
  • neben mobilen Plattformen wird auch die Skalierung auf Desktopsystemen ermöglicht

Anpassbares User Interface

  • Eingebauter Theming-Support
  • ThemeRoller Applikation zur einfachen Erstellung

Unterstütze Plattformen

A-Grade Apple iOS, Android, Windows Phone, Blackberry
B-Grade Symbian, Alte Versionen von Android, Blackberry und iOS
C-Grade Alte Versionen, die mal funktionierten aber nicht mehr getestet werden

Events

  • Touch Events: tap, tapholdswipe, swipeleft, swiperight
  • Device Orientation Events: orientationchange
  • Scroll Events: scrollstart, scrollstop
  • Page events: pagechange, pagetransition, pageinitialization

Coding mit JQuery Mobile

  • Webseite wird in Pages unterteilt
  • Ein div pro Page
  • Werden direkt geladen oder mit AJAX nachgeladen
  • Bootstrap-Code:
    <head>
<meta name="viewport" content="width=device-width, initial-scale=1">
<link rel="stylesheet" href="jquery.mobile-1.4.2.css">
<script src="jquery.js"></script>
<script src="jquery.mobile-1.4.2.js"></script>
</head>

Aufbau der HTML-Seite

<body>
<div data-role="page">

  <div data-role="header">
    <h1>Welcome To My Homepage</h1>
  </div>

  <div data-role="main" class="ui-content">
    <p>I Am Now A Mobile Developer!!</p>
  </div>

  <div data-role="footer">
    <h1>Footer Text</h1>
  </div>

</div>
</body>
